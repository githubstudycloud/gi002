## 🚀 使用场景

### 1. 克服写作瓶颈
**问题**: 不知道如何继续情节
**解决**:
- 输入当前情节
- AI提供多个发展方向
- 选择或融合建议
- 继续创作

### 2. 角色对话生成
**问题**: 对话不够生动
**解决**:
- 输入角色性格设定
- 描述当前场景
- AI生成符合性格的对话
- 人工调整润色

### 3. 场景描写
**问题**: 描写不够细腻
**解决**:
- 提供场景基本信息
- 指定描写风格（简洁/细腻）
- AI生成描写段落
- 融入故事中

### 4. 情节一致性检查
**问题**: 前后情节矛盾
**解决**:
- 上传全文或大纲
- AI分析情节线
- 标注矛盾之处
- 建议修改方案

## 💡 提示词工程技巧

### 基础提示词结构
```
角色: 你是一位专业的小说编辑
任务: 帮我优化这段对话
背景: 现代都市爱情小说
风格: 清新、细腻
内容: [要优化的文字]
要求: 保持人物性格，增强情感表达
```

### 高级提示词技巧

#### 1. Few-shot Learning
提供几个示例，让AI理解你的风格
```
示例1: [你的原文] → [期望效果]
示例2: [你的原文] → [期望效果]
现在请帮我处理: [新内容]
```

#### 2. 角色扮演
让AI扮演特定角色
```
你是一位擅长悬疑推理的作家，
请用你的专业眼光审视这段情节，
指出可能的逻辑漏洞和改进建议。
```

#### 3. 链式思考
引导AI逐步思考
```
请按以下步骤分析：
1. 分析角色动机
2. 梳理情节逻辑
3. 评估冲突设置
4. 提供优化建议
```

## 🎨 常用提示词模板

### 情节发展
```
当前情节: [描述]
主要角色: [列表]
情节目标: [目标]
请提供3个可能的发展方向
```

### 角色对话
```
角色A: [性格特点]
角色B: [性格特点]
场景: [具体场景]
冲突: [矛盾点]
请生成一段对话（约300字）
```

### 场景描写
```
地点: [位置]
时间: [时间段]
天气: [天气状况]
氛围: [期望氛围]
请生成场景描写（约200字）
```

### 大纲生成
```
小说类型: [类型]
主题: [核心主题]
主要角色: [列表]
关键情节点: [列表]
请生成30章的章节大纲
```

## 🔍 工具选择指南

### 按需求选择

**快速草稿** → GPT-4 + 简单提示词
- 快速生成内容
- 后期大幅修改

**高质量初稿** → Claude + 详细提示词
- 长文本理解好
- 质量较高
- 需要少量修改

**保持风格一致** → Fine-tuned模型
- 学习你的风格
- 高度定制化
- 需要训练数据

**隐私保护** → 本地LLM (Ollama)
- 完全私密
- 无网络依赖
- 需要硬件支持

### 按写作阶段选择

**规划阶段**: 大纲生成工具
- ainovelprompter
- 思维导图工具
- 情节卡片

**创作阶段**: 全功能写作助手
- WriterAI
- NovelAI
- Sudowrite

**润色阶段**: 编辑工具
- Grammarly (语法)
- ProWritingAid (风格)
- AI改写工具

## 🎯 最佳实践

### 1. 不要完全依赖AI
- AI是助手，不是替代品
- 保持个人创意和风格
- 人工审核和调整

### 2. 建立个人提示词库
- 收集有效的提示词
- 针对不同场景分类
- 持续优化改进

### 3. 迭代优化内容
- 第一版：快速生成
- 第二版：结构调整
- 第三版：细节润色
- 第四版：情感深化

### 4. 保持素材管理
- 角色卡片
- 世界观设定
- 情节时间线
- 重要物品/地点
