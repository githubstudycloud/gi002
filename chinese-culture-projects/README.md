## 📖 古诗词数据库

### 1. chinese-poetry/chinese-poetry ⭐ 41.9k+
**最全中华古诗词数据库**

- **仓库地址**: https://github.com/chinese-poetry/chinese-poetry
- **项目描述**: 最全面的中华古诗词数据库，JSON 格式存储
- **内容规模**:
  - 唐宋两朝近 14,000 位古诗人
  - 约 55,000 首唐诗
  - 约 260,000 首宋诗
  - 两宋时期 1,564 位词人
  - 21,050 首词作
- **数据集包含**:
  - 全唐诗
  - 全宋诗
  - 全宋词
  - 五代·花间集
  - 五代·南唐二主词
  - 论语
  - 诗经
  - 幽梦影
  - 四书五经
  - 蒙学
- **特点**: JSON 格式便于程序调用，已衍生出多个网页版、小程序、API 等应用

### 2. hlthu/Chinese-Poetry-Dataset
**中华古诗词数据集**

- **仓库地址**: https://github.com/hlthu/Chinese-Poetry-Dataset
- **项目描述**: 另一个全面的古诗词数据库项目
- **内容**: 与 chinese-poetry 类似的大规模诗词数据

## 📜 古典文学与古籍

### 1. luoxuhai/chinese-novel
**中国古典小说数据库**

- **仓库地址**: https://github.com/luoxuhai/chinese-novel
- **项目描述**: 📙 最全的中国古典小说数据库
- **内容**: 收录中国古典小说的数字化版本

### 2. shanleiguang/vBooks
**中文古籍数字复刻计划**

- **仓库地址**: https://github.com/shanleiguang/vBooks
- **项目全称**: Chinese Ancient Books Reproduction Project
- **项目特点**:
  - 数字化复刻中国古籍
  - 制作电子书版本复现原书外观
  - 提供专业工具支持
- **收录作品**:
  - 诗经
  - 楚辞
  - 四书
  - 陶渊明作品集
  - 等

### 3. Xunzi-LLM-of-Chinese-classics/XunziALLM
**荀子·古籍大语言模型**

- **仓库地址**: https://github.com/Xunzi-LLM-of-Chinese-classics/XunziALLM
- **项目描述**: 专门为处理古籍文本设计的大型语言模型
- **技术特点**:
  - 提供基础模型和对话模型
  - 基于 Qwen、ChatGLM、Baichuan 等框架
  - 针对古汉语文本优化

### 4. quzhi1/ChineseHistoricalSource
**中国历史资料搜索**

- **仓库地址**: https://github.com/quzhi1/ChineseHistoricalSource
- **项目描述**: 开源的中国历史资料搜索项目
- **功能**: 提供历史文献检索功能

### 5. GuwenBERT
**古文预训练语言模型**

- **主题标签**: literary-chinese
- **项目描述**: 专为文言文（古汉语）预训练的语言模型
- **应用**: 古文理解、翻译、分析等
