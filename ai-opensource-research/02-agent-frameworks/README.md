# AI Agent 框架集合

## 🏆 Top AI Agent 框架 (2025)

### 市场份额
- LangChain: 30%
- AutoGPT: 25%
- CrewAI: 20%

## 🌟 核心框架

### 1. LangChain
- **GitHub**: https://github.com/langchain-ai/langchain
- **市场地位**: 最广泛采用的Agent框架
- **核心特点**:
  - 模块化架构
  - 完整的LLM应用编排层
  - 支持多种工具和集成
  - LangSmith用于调试和优化
  - LangGraph用于复杂工作流

**使用场景**:
- 复杂的提示词链
- RAG应用
- 多步骤推理任务
- 工具调用和集成

### 2. CrewAI
- **GitHub**: https://github.com/joaomdmoura/crewAI
- **Stars**: 30,000+
- **核心特点**:
  - 多Agent编排
  - 角色化Agent系统
  - "团队"协作隐喻
  - 认证开发者社区

**使用场景**:
- 多角色协作任务
- 创意项目
- 需要不同视角的系统
- 复杂工作流协调

### 3. AutoGPT
- **GitHub**: https://github.com/Significant-Gravitas/AutoGPT
- **核心特点**:
  - 自主任务执行
  - 目标分解为子任务
  - 最小人工干预
  - 序列化执行

**使用场景**:
- 自动化研究
- 数据收集
- 多步骤任务自动化

### 4. AutoGen (Microsoft)
- **GitHub**: https://github.com/microsoft/autogen
- **核心特点**:
  - 多Agent对话框架
  - 可定制的对话模式
  - 人机协作
  - 代码执行能力

**使用场景**:
- 协作式问题解决
- 代码生成和调试
- 复杂决策系统

### 5. LangGraph
- **GitHub**: https://github.com/langchain-ai/langgraph
- **核心特点**:
  - 图状态机
  - 循环和条件逻辑
  - 持久化状态
  - 人机交互循环

**使用场景**:
- 复杂工作流
- 需要状态管理的应用
- 多路径决策系统
