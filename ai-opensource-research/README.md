# AI开源项目研究与资源搜集

本项目专注于探究和整理GitHub上流行的AI开源项目，包括各类工具、框架、应用和资源。

## 📁 目录结构

```
ai-opensource-research/
├── 01-prompts-collection/     # 提示词工程相关资源
├── 02-agent-frameworks/       # AI Agent框架和定义
├── 03-mcp-servers/           # Model Context Protocol服务器
├── 04-skills-and-hooks/      # Skills和Hooks定义
├── 05-multimedia-ai/         # 多媒体AI应用
│   ├── video-processing/     # 视频处理AI
│   ├── audio-processing/     # 音频处理AI
│   ├── image-processing/     # 图片处理AI
│   └── text-generation/      # 文本生成AI
├── 06-ai-applications/       # 综合AI应用
└── docs/                     # 研究文档和笔记
```

## 🎯 研究目标

1. **提示词工程** - 搜集高质量的提示词库、提示词工程工具和最佳实践
2. **Agent框架** - 研究主流的AI Agent框架、架构设计和实现方案
3. **MCP生态** - 探索Model Context Protocol的服务器实现和应用场景
4. **扩展机制** - 学习Skills、Hooks等AI系统扩展机制
5. **多媒体AI** - 收集视频、音频、图片、文本处理的AI工具
6. **实用应用** - 发现创新的AI应用案例和解决方案

## 📊 搜集标准

- ⭐ GitHub Stars > 1000
- 🔄 近期活跃更新（6个月内）
- 📝 完善的文档
- 🔓 开源协议友好
- 💡 创新性和实用性

## 🔍 搜集方法

1. GitHub Trending搜索
2. GitHub Topics筛选
3. Awesome Lists汇总
4. 社区推荐
5. 技术博客和论文
